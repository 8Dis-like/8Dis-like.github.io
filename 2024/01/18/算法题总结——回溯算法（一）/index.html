1. 回溯法

回溯是一种搜索方式，本质是穷举。使用回溯法解题有以下三个要素要明晰：

  • 回溯函数的参数及返回值

  • 回溯函数的终止条件

  • 单层循环所要执行的操作

    这里附上此三个步骤的伪代码加深理解：

void backtracking(参数) {
    if (终止条件) {
        存放结果;
        return;
    }

    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果
    }
}

下面的解题思路基本按照此三个步骤展开。

2. 题目列表

2.1. 组合问题

给定两个整数 n 和 k，返回 1 … n 中所有可能的 k 个数的组合。

示例: 输入: n = 4, k = 2 输出: [ [2,4], [3,4], [2,3], [1,2], [1,3], [1,4], ]

这道题可以直接用两层签到循环解决，但是如果k=100那就要写100层嵌套循环，这显然不可能，所以考虑使用回溯法，回溯的本质还是循环遍历。

  1. 参数部分，题干明显给出两个参数，这是不可缺少的。另外在回溯算法中，因为每次的回溯都是从上一次开始，所以一般都会有一个记录循环开始位置的startindex作为参数。

  2. 终止条件就是得到的每个结果中的数字个数等于要求的k。

  3. 单层循环内容是把每次遍历得到的数组加入结果中，知道达到循环终止条件，存储结果。

以下是回溯的具体代码，结合具体代码再来看这个回溯过程。

vector<vector<int>> results; // 存放符合条件结果的集合
vector<int> path; // 用来存放符合条件结果
void backtracking(int n, int k, int startindex){
    if(path.size() == k){
        result.push_back(path);
        return;
    }
    for(int i = startindex; i < n; i++){
        path.push_back(i+1);
        backtracking(n, k, i+1);
        path.pop_back();
    }
}

整个回溯过程首先从startindex=0开始，首先第一次进入循环，i=0，执行循环内代码：存储第一个遍历到的值，当执行到循环内的backtracking函数时，便开始了递归。第一次递归传入的startindex=1，那么接下来便会存储第二个遍历到的值，然后再次进入递归。再次进入递归后发现path中已经存储了两个数了，所以此次回溯结束。下面再进入的时第一次递归后传入的startindex=1的下一步，也就是startindex=2，以此类推，直至startindex=4，第一次递归才算结束。然后i=1，重复刚才的过程。

另外回溯函数后，还将刚加入结果的pop了。这里的pop其实是当我们找到一个合适的结果后，并且将其存储后，再依次把path中的值删除，供下一循环重新使用。

从上面过程可以看出，回溯是先往深处遍历，满足条件后结束，并跳转至最近的递归层。for循环其实是每层中每个节点处开始横向遍历的过程，而嵌套则是层数深入的过程,整个结构与树的结构类似。

2.2. 组合求和（一）

找出所有相加之和为 n 的 k 个数的组合。组合中只允许含有 1 - 9 的正整数，并且每种组合中不存在重复的数字。
示例 1: 输入: k = 3, n = 7 输出: [[1,2,4]]
说明：

  • 所有数字都是正整数。
  • 解集不能包含重复的组合。

这题与上一道题相似，上题是任意组合，本题要求和为n。这里还是考虑三个要素：

  1. 回溯函数的参数：n和k，还有startindex，这里是求和，最好是在传入一个参数sum，每遍历到一个数就相加并判断是否符合条件。
  2. 终止条件是结果中的数字个数等于k，并且此时sum为n。
  3. 单层循环内执行的是sum加上此次遍历得到的数。

根据以上分析可以写出以下回溯代码

vector<vector<int>> results; // 存放符合条件结果的集合
vector<int> path; // 用来存放符合条件结果
void backtracking(int n, int k, int startindex,int sum){
	if (sum > targetSum) { // 剪枝操作
        return; 
    }
    if(path.size() == k && sum == n){
        result.push_back(path);
        return;
    }
    for(int i = startindex; i <= ; i++){
        path.push_back(i);
        sum += i;
        backtracking(n, k, i+1, sum);
        path.pop_back();
        sum -= i;
    }
}

2.3. 组合求和（二）

给定一个无重复元素的数组 candidates 和一个目标数 target ，找出 candidates 中所有可以使数字和为 target 的组合。candidates 中的数字可以无限制重复被选取。

说明：
所有数字（包括 target）都是正整数。
解集不能包含重复的组合。

示例：
输入：candidates = [2,3,6,7], target = 7 输出： [ [7], [2,2,3] ]

本题条件限制增加，但还是先考虑三要素，再讨论细节。

  1. 回溯函数的参数：预选数组candidates和求和目标数target，同样还有startindex和sum。
  2. 终止条件是sum=target。
  3. 单层循环内执行的是sum加上此次遍历得到的数。

此题明确说明数字可以重复使用，如果按照上面的思路，选取一个数后便从其下一个数继续回溯，那就没有考虑到重复选取，所以此处回溯的startindex不能再使用i+1，而应该是i。

vector<vector<int>> results; // 存放符合条件结果的集合
vector<int> path; // 用来存放符合条件结果
void backtracking(vector<int> candidates, int target, int startindex,int sum){
	if( sum == target){
           result.push_back(path);
           return;
	}
	for(int i = startindex; i < candidates.size(); i++){
           path.push_back(i);
           sum += i;
           backtracking(n, target, i, sum);
           path.pop_back();
           sum -= i;
	}
}

2.4. 组合求和（三）

给定一个数组 candidates 和一个目标数 target ，找出 candidates 中所有可以使数字和为 target 的组合。candidates 中的每个数字在每个组合中只能使用一次。

说明： 所有数字（包括目标数）都是正整数。解集不能包含重复的组合。

输入: candidates = [10,1,2,7,6,1,5], target = 8,
所求解集为: [[1, 7],[1, 2, 5],[2, 6],[1, 1, 6]]

此题相较上题又有变化，这里没有限制所给数组无重复元素，但同时限制解集不能有重复组合。

这里的三要素其实与上面的是一样的，不同的是数组有重复元素，但结果不能有重复，那么遍历过程肯定需要有调整。

我们研究遍历细节，假如所给数组为[1，1，2，3],target为3第一层就会取到两次1，第一次取1后，随着层数深入，后续会取到2，这就是一个正确的结果。而第二次取1后，后续还会取到2，这样就会由于重复元素出现重复结果。要避免这种情况，那么就要保证同层中第一次取1后，就不再取第二个1。

要想实现上面的筛选，首先要对数组进行排序，再执行回溯前判断是否有相同的元素并且第一个已被使用的情况，若有则执行continue，修改代码如下：

vector<int> path;
   vector<vector<int>> results;
   void backtracking2(vector<int> candidates, int target, int sum, int startIndex)
   {
       if (sum == target)
       {
           results.push_back(path);
           return;
       }
       for (int i = startIndex; i < candidates.size() && sum + candidates[i] <= target; i++)
       {
           if (i > startIndex && candidates[i] == candidates[i - 1])
           {
               continue;
           }
           sum += candidates[i];
           path.push_back(candidates[i]);
           backtracking2(candidates, target, sum, i + 1);
           sum -= candidates[i];
           path.pop_back();
       }
   }

2.5. 电话号码组合

给定一个仅包含数字 2-9 的字符串，返回所有它能表示的字母组合。

给出数字到字母的映射如下（与电话按键相同）。注意 1 不对应任何字母。

示例:
输入：”23”
输出：[“ad”, “ae”, “af”, “bd”, “be”, “bf”, “cd”, “ce”, “cf”].

首先看三要素：

  1. 回溯函数的参数：所给的字符串，以及startindex。
  2. 终止条件是结果的大小等于所给字符串的大小。
  3. 单层循环内执行的是遍历每个数字所对应的字母，并且添加到结果中。

单层循环所执行的内容具体来说：首先找到对应的数字存储起来（由于给的是字符串数字，所以要把类型转为int型），然后找到这个数字对应的字母字符串，然后遍历字符串，存储结果。

这里需要注意的是，在执行横向遍历时，我们遍历的是每个数字对应的字母组合。纵向遍历也就是层数深入才是切换下一个数字，所以循环中的i是从0开始，而不是从startindex开始。下面是主要思路的代码：

vector<int> path;
   vector<vector<int>> results;
void backtracking(string &digits, int startindex){
	if(path.size() == digits.size()){
		results.push_back(path);
		return;
	}
	int digit = digits[startindex] - '0';
	string letters = lettermap[digit];
	for(int i = 0; i < letters.size(); i++){
		path.push_back(letters[i]);
		backtracking(digits, startindex+1);
		path.pop_back();
	}
}

2.6. 切割回文串

给定一个字符串 s，将 s 分割成一些子串，使每个子串都是回文串。

返回 s 所有可能的分割方案。

示例: 输入: “aab” 输出: [ [“aa”,”b”], [“a”,”a”,”b”] ]

三要素：

  1. 回溯函数的参数：所给的字符串，以及startindex。
  2. 终止条件是startindex大于或等于字符串的长度。
  3. 单层循环内执行的是从startindex开始，往后依次选取字符串，并判断是否为回文串。
bool IsPalindrome(string &input);
void backtracking(string &input, int startindex){
	if(startindex >= input.size()){
		result.push_back();
		return;
	}
	for(int i = startindex; i < input.size();i++){
		if(IsPalindrome(input, startindex, i)){
			string str = input.substr(startindex, i - startindex + 	1);
			path.push_back(str);
			backtracking(input, i + 1);
			path.pop_back();
		}else{
			continue;
		}
	}
}

难点在于理解切割的过程并抽象为回溯算法来解决，也就是利用单层循环中startindex和i依次选取字符串进行回文判断，并存储结果。如果把回溯问题看作树结构，从横向和纵向的遍历理解清楚，或许更容易找到单词循环需要做什么，从而找到解决问题的思路。

另外，判断回文时，下面代码中有两种方法，对比前者更慢，后者更快，也就是说c++库函数的执行效率有时会更慢，如果对执行效率等有要求时，需要谨慎使用库函数。

bool IsPalindrome(string &s)
{

    // string revstr = s;
    // reverse(s.begin(), s.end());
    // return !s.compare(revstr);

    for (int i = 0, j = s.size() - 1; i < j; i++, j--)
    {
        if (s[i] != s[j])
        {
            return false;
        }
    }
    return true;
}

2.7. 分割IP地址

给定一个只包含数字的字符串，复原它并返回所有可能的 IP 地址格式。

有效的 IP 地址 正好由四个整数（每个整数位于 0 到 255 之间组成，且不能含有前导 0），整数之间用 ‘.’ 分隔。

示例 1：
输入：s = “25525511135”
输出：[“255.255.11.135”,”255.255.111.35”]

观察输出，其实相比于输入就是通过”.”进行了分割，所以本题可以理解为在所给字符串中插入三个”.”，使得生成的新字符串符合IP地址的格式。

三要素：

  1. 回溯参数：输入字符串，startindex，以及记录已经插入的点数dotnum
  2. 终止条件：当已经插入三个点时，第四段就被分割好了，故终止为插入第三个点。
  3. 单层执行内容：遍历数组，并且判断所取数段是否符合IP地址要求，符合则在该数后插入点。
void backtracking(string &digits, int startindex, int dotnum){
	if(dotnum == 3){
		if(IsVaild(digits, startindex, digits.size() - 1)){
			result.push_back(path);
		}
		return;
	}
	for(int i = startindex; i < digits.size(); i++){
		if(Isvalid(digits, startindex, i)){
			digits.insert(digits.begin() + i + 1, '.');
			dotnum++;
			backtracking(digits, i + 2, dotnum);
			digits.erase(digits.begin() + i + 1);
		}else{
			break;
		}
	}
}
bool IsValid(string &str, int begin, int end){
	if( end < begin || end - begin >= 3){
		return false;
	}
	if(str[begin] == '0' && start != end){
		return false;
	}
	int num = 0;
	for(int i = begin; i <= end; i++){
		num = num*10 +(str[begin] - '0');
		if(num > 255){
			return false;
		}
	}
}

本题的难点在于明晰点的位置选取的遍历过程，进而截取IP段并判断是否合法。同样可以将其抽象为树形结构，纵向遍历相当于提供截取位置，横向遍历就是随着i的移动截取[startindex，i]中的多个数字。

另外之前的回溯中不合法情况都是continue，但这里却是break，因为一旦截取[startindex,i]不合法，那么i在往后移动也一定不合法，所以直接跳出循环，而不是仅仅不执行此次循环。
