上面一共是四个不同类型的可视化方法

  • countplot：这是一种创建计数柱状图的函数，纵轴表示对应类别的观测数，每个柱子的颜色可以根据不同类型进行区分。
  • histplot：这是一种创建直方图的函数，可以将数据分为多个连续区间，并且计算每个区间的评述或者概率.
  • pointplot：这是一种绘制线状图的函数,可以表示类别数据与数值数据间的关系，对对应类别的数值数据取均值
  • barplot：这是一种给绘制柱状图的方法，作用与pointplot类似。

# 这里使用Sex和Pclass两个属性来判断空缺年龄的值并填补
filled_age = np.zeros((2, 3))
for dataset in combine:
    for i in range(1, 3):
        for j in range(1, 4):
            ages = dataset[(dataset['Sex'] == i)
                           & (dataset['Pclass'] == j)]['Age'].dropna()
            filled_age[i - 1, j - 1] = ages.median()
    for i in range(1, 3):
        for j in range(1, 4):
            dataset.loc[(dataset.Sex == i) & (dataset.Pclass == j) &
                        (dataset.Age.isnull()), 'Age'] = filled_age[i - 1,
                                                                    j - 1]
    dataset['Age'] = dataset['Age'].astype(int)
